High-level task: Consume -> Discover

Because the data in the visualization is presented to let us make discoveries about the daily maximum temperatures over time.

Medium-level task: Search -> Lookup

Because the visualization provides both the target and location through the 3 highlighted points

Low-level task: Query -> Identify

Because the cluster of highlighted points are the singular target

Target(s): Outlier

Because the highlighted points are an outlier to the rest of the plot

High-level task: Consume -> Present

Because the purpose of the visualization is to show the times daily mean temparture was 20 degrees above normal in both cities and allow us to immediately see the similarities and differences rather than make comparisons

Medium-level task: Search -> Browse

Because we know the location (Seattle, Portland) but we don't know the target

Low-level task: Query -> Compare

Because you are comparing the temperature plots of Seattle and Portland

Target(s): Similarity

Because the visualization is comparing the number of times daily mean temperatures were 20 degrees Fahrenheit above normal in both Seattle and Portland during the same time period to find similarities in the cities' data
